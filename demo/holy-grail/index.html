The Holy Grail

This illustrates a Holy Grail design variant, containing:

  • One header.
  • One footer.
  • A "centerpiece" - an area for central content, central as in important, which coincides, visually, with central (i.e. not peripheral) placement: The container of this very text.

Note how this pattern scales by simple repetition on the horizontal or vertical:

The sidebars have exactly the same structure as the main page container: header, centerpiece, footer.
